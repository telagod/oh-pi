# Demo Assets Checklist (2-Minute Aha)

Use this checklist before publishing the 2-minute demo.

## 1) Recording Assets
- [ ] `demo.cast` (asciinema raw recording)
- [ ] `demo.gif` (short preview for README/social)
- [ ] `demo.mp4` (optional, for platforms requiring video)

## 2) Script Alignment
- [ ] Demo flow matches `docs/DEMO-SCRIPT.md`
- [ ] Includes setup moment (`npx oh-pi` -> `pi`)
- [ ] Includes colony signal progression (`SCOUTING` -> `COMPLETE`)
- [ ] Explicitly states when **not** to use colony

## 3) Repo Artifacts
- [ ] Link demo from README top section
- [ ] Add one screenshot under `docs/` (optional)
- [ ] Confirm no secrets/API keys appear in recording

## 4) Publishing Checklist
- [ ] Upload asciinema and verify playback speed
- [ ] Publish short intro text (problem -> setup -> value -> boundary)
- [ ] Share in first target community channel
- [ ] Collect first 5 external feedback comments/issues
